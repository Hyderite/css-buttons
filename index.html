CSS Buttons



1. From black to white





Code:


         
           .btn1 {
              background-color: #000;
              border: none;
              color: white;
              text-align: center;
              text-decoration: none;
              display: inline-block;
              font-size: 16px;
              font-family: /* the font of your choice */;
              margin: 4px 2px;
              -webkit-box-shadow: 3px 3px 14px 0px rgba(0, 0, 0, 0); 
              box-shadow: 3px 3px 14px 0px rgba(0, 0, 0, 0);
              border-radius: 20px;
              width: 125px;
              height: 40px;
              cursor: pointer;
              transition: 0.3s;
            }

            .btn1:hover {
              background-color: #fff;
              color: #000;
              border: 4px solid black;
            }


2. Darker on hover





Code:


         
           .btn2 {
              background-color: #ff0000;
              border: none;
              color: white;
              text-align: center;
              text-decoration: none;
              display: inline-block;
              font-size: 16px;
              font-family: /* the font of your choice */;
              margin: 4px 2px;
              -webkit-box-shadow: 3px 3px 14px 0px rgba(0, 0, 0, 0); 
              box-shadow: 3px 3px 14px 0px rgba(0, 0, 0, 0);
              border-radius: 8px;
              width: 125px;
              height: 40px;
              cursor: pointer;
              transition: 0.3s;
            }

            .btn2:hover {
              filter: brightness(0.8);
            }


3. Hover Gradient





Code:


         
           .grad-btn {
              background-image: linear-gradient(to bottom left, #743ad5, #d53a9d);
              border: none;
              color: white;
              text-align: center;
              text-decoration: none;
              display: inline-block;
              font-size: 16px;
              font-family: Montserrat;
              margin: 4px 2px;
              box-shadow: 3px 3px 14px 0px rgba(0, 0, 0, 0);
              border-radius: 8px;
              width: 125px;
              height: 40px;
              cursor: pointer;
              transition: 0.3s;
              animation: fadein 1.5s;
              animation-delay: 3.5s;
              opacity: 0;
              animation-fill-mode: forwards;
            }

            .grad-btn:hover {
              transform: scale(1.1);
            }
